## If anyone asks (Q&A backups)

**"Why passkey not seed phrase?"** Seed phrases are a UX disaster for non-crypto-native users. Passkey is biometric, it's already in their iCloud or Google account, it's recoverable. Same security guarantees with a tenth of the friction.

**"Why mobile first?"** Because the audience we built for already lives on their phone. Lagos freelancer getting paid in USDC isn't opening a browser extension to manage it.

**"How does private send actually work?"** Umbra SDK handles the cryptography. We surface it as a toggle. User sends, recipient claims encrypted UTXO, on-chain it looks like noise. Recipient doesn't have to be a Seedless user, just needs to be registered with Umbra.

**"When can I try it?"** Private mainnet beta is live now, public launch end of June. Beta access link in our TG, link on seedlesslabs.xyz.
